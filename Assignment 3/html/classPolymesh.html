void Polymesh::generateMesh ( std::vector< Point2 >  points,
int  slices 
)

Function to generate the Polymesh from a set of 2D points in xy coordinate system.

It samples the points vector for every 50th point, and finds the set of points around the axis of revolution.

Parameters
points: Vector of points(pixels on the curve)
slices: number of slices along the y axis, which determines the angle of rotation for each point generation.
