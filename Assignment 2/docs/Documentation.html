Computer Graphics- Assignment 2

3-D modelling in simple words is creating a scene as we visualise and place it on a screen using basic primitives avialable in graphics which undergo various tranformations such that the end result is the desired object. A camera interacts with the scene with various movements available such as rotate, translate, pan, zoom etc.

To create a scene depicting a children' park; following tasks were completed:-
Task 1: Using openGl primitives following elements were created in the park:

  • See saw
  • Jungle Gym
  • Monkey Bars
  • Slide
  • Swings
  • Merry-go Round
  • Benches
  • Boundary walls

Task 2 : Blender was used to include different types of low- poly trees and stones. 4 different kinds of trees were generated.



Task 3 : Following interactive functionality was implemented 
       		Keyboard inputs:

       		w,upkey: forward 

       		s,downkey: backwards 

       		a,leftkey: leftwards 

       		d,rightkey: rightwards 



       		r: rotate

       		y: yaw, shift+y: reverse direction

       		p: pan, shift+p: reverse direction

       		q: move upwars

       		z: move downwards



       		LMB: zoom-in

       		RMB: zoom-out
